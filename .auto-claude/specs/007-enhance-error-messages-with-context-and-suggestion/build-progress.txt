# Build Progress: Enhance Error Messages with Context and Suggestions

- What went wrong
- Why it happened
- How to fix it

Uses consistent color coding and formatting inspired by CLI tools like rustc, gh, and kubectl.

## Key Implementation Decisions

### Error Structure
```go
type WeaverError struct {
    Code        ErrorCode
    Message     string
    Context     map[string]string
    Cause       error
    Suggestions []string
}
```

### Color Scheme
- Red (#E74C3C) - Error type/code
- Yellow (#F39C12) - Context information
- Cyan (#3498DB) - Suggestions
- Green (#27AE60) - Success hints

### Output Format
```
ERROR [ConfigNotFound]: Configuration file not found
  Path: ~/.config/weaver/config.yaml
  Suggestion: Run 'weaver --init' to create a default configuration
```
